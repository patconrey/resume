AR Wayfinding

This is a project I built for my employer, Gamifi.

Gamification encompasses a broad range of technologies. One particularly compelling technology on the horizon is augmented reality, AR. We wanted to create a way that a client could define landmarks on a map and we can show users those landmarks in the real world.

Since Gamifi has a number of clients that want to implement this feature, I built it as a modular framework with intuitive data source and delegate protocols so we can easily implement the framework in the future.
